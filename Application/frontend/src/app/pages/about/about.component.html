Gemini 2.5 Flash

Context-locked conversational AI assistant that provides actionable recommendations based exclusively on model predictions and KG outputs.

Real-time animated pipeline processing entire image directories with comprehensive MongoDB logging, ETA tracking, and summary dashboards.
